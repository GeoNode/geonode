Upload data layer!

You can upload your data layer on following formats on GeoDASH platform:

Shape Layer : .shp .shx .dbf .prj
Zipped Shape Layer : .shp .shx .dbf .prj
Open Street Map Layer : .osm
Excel Format Data Layer : .csv
Image Layer : .tif

Create a new map!

You can create your own map using single or multiple data layers from GeoDASH platform. You can also create map using data layer from other geonode/openstreet/ server that supports WMS request based on your access level. You have the ability to edit/ update the style of your map.

Upload document!

You can upload your data layer documents on GeoDASH platform and link it with the desired layer or map

Allowed document types:
.doc .docx .gif .jpg .jpeg .ods .odt .odp .pdf .png .ppt .pptx .rar
.sld .tif .tiff .txt .xls .xlsx .xml .zip .gz

Download layer!

You can download any public layer or permitted privet layer(s) from GeoDASH platform.

Allowed download types are:
KML, GeoJSON, Excel, CSV, GML 3.1.1, GML
2.0, Zipped Shapefile, PNG, PDF, JPEG

Explore organizations!

You can explore the list of registered organizations in GeoDASH platform and view the public layer list, User list and ask question to that organization admin.

Explore users!

You can explore the list of registered users of GeoDASH platform after login.

Dock your favorite items!

You can dock you favorite layers, maps, documents and organizations for quick access and to follow the activity.

Messaging!

You can communicate with any other users of GeoDASH system using message option for your requirement.
