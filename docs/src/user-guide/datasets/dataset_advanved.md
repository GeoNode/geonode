## Advanced Dataset Management with MapStore { #dataset-advanced }

GeoNode provides the user with some advanced features for dataset manipulation such as dataset filtering, attribute edition and dataset export in different formats.

In a nutshell, these features are provided via MapStore and the user will be redirected to MapStore-specific documentation for further details.

### Filtering Datasets

With GeoNode you can filter a dataset via its attributes, directly on the map by drawing an area of interest over the map canvas, and via cross-dataset filter, allowing intersection, contained and contains overlay methods.

For more details please check the MapStore documentation [here](https://mapstore.readthedocs.io/en/latest/user-guide/filtering-layers/).
