Managing Maps
=============

| Maps are sets of layers displayed together on an interactive web map.
  Maps can be composed in the map composer and saved as GeoNode resources.
  Maps can also be associated with metadata, ratings, and comments.
| In this section, you will learn how to create a new map and share it.
